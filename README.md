# 🤖 Advanced Telegram AI Bot & Media Downloader

Многофункциональный асинхронный Telegram-бот и юзербот на Python. Совмещает в себе локальный ИИ-ассистент, систему распознавания медиа (голосовые, картинки) и продвинутый инструмент для автоматизации, мониторинга и скачивания контента из закрытых источников.

## 🔌 Доступные команды в боте

Поскольку команды не зарегистрированы в стандартном меню Telegram, их необходимо вводить вручную. Доступ к основному функционалу строго ограничен по `admin_id`.

*   `/start` — Команда для обычных пользователей. Выводит сообщение, что бот находится в технической обработке.
*   `/vakson` — **Основная команда запуска для администратора**. Открывает интерактивную админ-панель управления сессиями, авторизации аккаунтов, мониторинга и скачивания.
*   `/reload_plugins` — Динамическая перезагрузка всех плагинов и модулей на лету без отключения бота.
*   `/ai` — Запрос к локальному ИИ-ассистенту через Ollama.
*   `/pin` — Парсинг и скачивание медиаконтента из Pinterest.
*   `/youtube` — Загрузка видео или аудио из YouTube через встроенный модуль `yt-dlp`.
*   `/analyze` — Запуск глубокого анализа изображений или логов.
*   `/generate` — Генерация контента / изображений с помощью подключенных нейросетей.

## 🔥 Ключевой функционал

*   **🔌 Модульная архитектура:** Поддерживает плагины с динамической перезагрузкой.
*   **🧠 Локальный ИИ (Ollama):** Текстовые ответы и генерация кода прямо в чате.
*   **🎙 Распознавание речи (Whisper):** Автоматическая транскрипция входящих голосовых сообщений.
*   **👁 Анализ изображений:** Обработка визуального контента нейросетями.
*   **🔑 Авторизация аккаунтов (Userbot):** Необходима для полноценного мониторинга каналов и обхода ограничений на скачивание.
*   **📂 Слив закрытых каналов:** Парсинг фото/видео из источников, где админы запретили копирование и сохранение контента.
*   **🗄 Семантический поиск и БД:** Хранение данных в SQLite с использованием SQLAlchemy.

🖥 **Рекомендуемые системные требования:**
*   **Видеокарта (GPU):** Дискретная графика с объемом видеопамяти от **6 ГБ VRAM** (тестировалось на *GTX 1080 Ti 11GB*).
*   **Важное примечание по памяти:** Каждая локальная модель (Qwen, Moondream) в работе потребляет около **3–5 ГБ VRAM**.
*   **Рекомендации:** Рекомендуется использовать функции по отдельности. Одновременная пиковая нагрузка на ИИ-модули может потребовать больше памяти**.

## 📦 Установка и настройка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/rose-logan/telegram-ai-bot
   cd telegram-ai-bot
   ```

2. Установите все необходимые библиотеки:
   ```bash
   pip install -r requirements.txt
   ```

3. **Медиа-зависимости:** Для работы модулей обработки аудио/видео скачайте исполняемые файлы `ffmpeg.exe` и `ffprobe.exe` и положите их прямо в корневую папку проекта*.
   **Загрузка локальных ИИ-моделей (Ollama):**
   Убедитесь, что у вас установлено и запущено приложение [Ollama](https://ollama.com). Для автоматической загрузки необходимых моделей (`qwen2.5:1.5b` и `moondream`), запустите скрипт автоматизации в терминале PowerShell:
   ```powershell
   ./install_models.ps1
   ```
   *(Или просто запустите файл `install_models.ps1` двойным кликом мыши).*
4. **Настройка конфигурации (Критично):** 
   Для работы бота необходимо отредактировать файл `config.py`. Внесите в него свои данные авторизации:
   *   `API_ID` и `API_HASH` (полученные на my.telegram.org)
   *   `BOT_TOKEN` (от BotFather)
   *   `ADMIN_ID` — ваш цифровой ID в Telegram. Все ключевые команды бота будут выполняться строго от имени этого администратора.

5. Запустите проект:
   ```bash
   python main.py
   ```

## ⚠️ Важное примечание по безопасности
Файлы сессий (`*.session`), базы данных (`*.db`) и исполняемые файлы (`ffmpeg.exe`, `ffprobe.exe`) добавлены в `.gitignore` и не публикуются в открытый доступ для защиты конфиденциальных данных учетных записей.
