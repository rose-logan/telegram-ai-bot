# 🤖 Advanced Telegram AI Bot & Media Downloader

Многофункциональный асинхронный Telegram-бот и юзербот на Python. Совмещает в себе локальный ИИ-ассистент, систему распознавания медиа (голосовые, картинки) и продвинутый инструмент для автоматизации, мониторинга и скачивания контента из закрытых источников.

## 🔥 Ключевой функционал

*   **🔌 Модульная архитектура:** Поддержка плагинов с возможностью динамической перезагрузки прямо на лету через команду `/reload_plugins` без перезапуска самого бота.
*   **🧠 Локальный ИИ (Ollama):** Интеграция с ИИ-моделями для текстовых ответов и работы с кодом напрямую через чат.
*   **🎙 Распознавание речи (Whisper):** Автоматическая транскрипция входящих голосовых сообщений в текст.
*   **👁 Анализ изображений:** Обработка и распознавание визуального контента с помощью нейросетей.
*   **🔑 Авторизация аккаунтов (Userbot):** Встроенный клиент для авторизации Telegram-аккаунтов через сессии. Авторизация необходима для полноценного мониторинга каналов и скачивания медиа.
*   **📂 Обход ограничений каналов:** Парсинг и скачивание оригиналов фото/видео из закрытых Telegram-каналов, где администраторы запретили копирование и сохранение контента.
*   **🗄 Семантический поиск и БД:** Хранение данных, логов и настроек в SQLite с использованием SQLAlchemy и механизмов семантического поиска.

## 🛠 Технологический стек

*   **Язык:** Python 3.10+
*   **Библиотеки ИИ:** Ollama, OpenAI Whisper, Pillow (PIL)
*   **ТГ-API:** Telethon / Pyrogram / Aiogram
*   **База данных:** SQLite, SQLAlchemy
*   **Зависимости:** Управление через `requirements.txt`

## 📦 Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/rose-logan/telegram-ai-bot
   cd telegram-ai-bot
   ```

2. Установите все необходимые библиотеки:
   ```bash
   pip install -r requirements.txt
   ```

3. **Важно (Медиа):** Для работы модулей обработки аудио/видео скачайте исполняемые файлы `ffmpeg.exe` и `ffprobe.exe` и положите их прямо в корневую папку проекта.

4. Настройте конфигурацию:
   Переименуйте файл настроек (или отредактируйте `config.py`), указав туда свои `API_ID`, `API_HASH` и `BOT_TOKEN`.

5. Запустите проект:
   ```bash
   python main.py
   ```

## ⚠️ Важное примечание по безопасности
Файлы сессий (`*.session`), базы данных (`*.db`) и исполняемые файлы (`ffmpeg.exe`, `ffprobe.exe`) добавлены в `.gitignore` и не публикуются в открытый доступ для защиты конфиденциальных данных учетных записей.
