# telegram-ai-bot
# 🤖 Telegram AI Bot

Краткое описание в одно предложение: что делает бот? (Например: Асинхронный Telegram-бот с интеграцией Ollama/ИИ для автоматизации ответов и умного поиска).

## 🚀 Функционал
*   **ИИ-ассистент:** Интеграция с локальными моделями через Ollama (`ai_handler.py`, `ai_router.py`).
*   **Семантический поиск:** Быстрый поиск по базе данных (`semantic_search.db`).
*   **Управление клиентами:** Модуль `client_manager.py` для контроля сессий.
*   **Поддержка прокси:** Безопасная работа через `proxy.py`.

## 🛠 Технологический стек
*   **Язык:** Python 3.10+
*   **База данных:** SQLite / SQLAlchemy (`database.py`, `bot_data.db`)
*   **ИИ-движок:** Ollama / [moondream][qwen2.5:1.5]

## 📦 Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com](https://github.com/rose-logan/telegram-ai-bot/
   cd telegram-ai-bot
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Настройте конфигурацию:
   Переименуйте или создайте файл конфигурации (укажите, где менять токен бота и ключи, например, в `config.py` или `.env`).

4. Запустите бота:
   ```bash
   python main.py
   ```
